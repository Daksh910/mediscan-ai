\# MediScan AI — Clinical Intelligence Platform



Full-stack diabetes risk assessment platform powered by ensemble machine learning.



\## Tech Stack

\- \*\*Backend:\*\* Django REST Framework + PostgreSQL (Supabase)

\- \*\*ML Engine:\*\* Stacking Ensemble (Random Forest + XGBoost + LightGBM + Gradient Boosting)

\- \*\*ML Performance:\*\* ROC-AUC: 0.83 | F1: 0.78 | Recall: 0.88

\- \*\*Frontend:\*\* React + TypeScript + Vite + Recharts

\- \*\*Auth:\*\* JWT Authentication + Password Reset via Email



\## Running Locally



\### Backend

```

cd backend

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver

```



\### Frontend

```

cd frontend

npm install --legacy-peer-deps

npm run dev

```
